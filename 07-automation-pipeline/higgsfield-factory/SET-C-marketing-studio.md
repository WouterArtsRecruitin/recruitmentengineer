# SET C — Marketing Studio Ads

8 ad-format-specific creative templates for paid distribution. 2 prompts per format × 4 formats. Color grade locked to brand DNA.

> **Tool:** Higgsfield Marketing Studio (uses Soul 2 character lock + ad-format presets). Render at native ad-platform aspect ratios.

> **Pre-flight:** read `identity-lock-rules.md`. Brand colors: `#FF6B1A` (orange), `#1E3A5F` (blueprint blue), `#00D4FF` (AI cyan, sparingly), `#F5F0EB` (warm grey), `#1A1A1A` (deep black).

> **Locked color grade across all 8 ads:** teal-orange cinematic, Recruitin orange in highlights, blueprint-blue in shadows. Same lens character (50–85mm, f/2.0–f/2.8).

## Format 9:16 — Reel (Meta / Instagram / TikTok / LinkedIn vertical)

### C.1 — Reel hook-card (typography-led, no portrait)

```
Marketing Studio — 9:16 (1080×1920) — Reel

Vertical hook card design. Background: pure deep black #1A1A1A with
extremely subtle blueprint grid pattern at 4% opacity.

Top third: blank breathing room.

Middle third — massive stacked headline, all caps, Inter Bold:
  "GEEN VACATURES." (white, 110pt)
Below in orange #FF6B1A:
  "EEN SYSTEEM." (110pt)

Below the headline, 12px high orange #FF6B1A rule, 200px wide.

Lower third — supporting text, Inter Regular 36pt warm grey #F5F0EB:
"5 prompts. 47 getest. Gratis PDF."

Bottom 12% safe zone — clear of all type (Reels UI overlay zone).

Subtle vignette fade to deeper black at top + bottom edges.

Engineered for thumb-stop on Reels feed. Reads at 400px preview.

NEGATIVE: misspelled text, decorative ornaments, drop shadows,
gradients, glow, cartoon, generic motivational style, AI artifacts,
watermark, busy background.
```
