  - Scrapes LinkedIn profile → genereert geïndividualiseerde InMail

### **Module 3 — Intake Mastery (p8-9)**
- **Tool stack:** Notion intake template + Pipedrive workflow + Loom recordings
- **AI prompt 3:** "Vacature Intake Generator"
  - Input: 30-min klantgesprek transcript
  - Output: complete vacature-tekst + USP's + ICP profile + sourcing-strategie
  - **Real example:** Beutech — intake → live vacature in 4 uur (was 3 dagen)
- **Tijd-besparing:** 8 uur intake-werk → 90 minuten

### **Module 4 — Kandidaat-Kwalificatie (p10)**
- **Tool stack:** Custom screening forms (Jotform) + AI scoring
- **AI prompt 4:** "Match Score Calculator"
  - Input: vacature-tekst + CV
  - Output: match score (0-100) + gap analysis + interview-vragen op maat
  - **Real example:** 73% accuracy vs. handmatige beoordeling (gemeten over 200 kandidaten)
- **Tijd-besparing:** 20 min/CV → 4 min/CV

### **Module 5 — Tracking & Optimization (p11)**
- **Tool stack:** GA4 + Pipedrive analytics + custom dashboards
- **AI prompt 5:** "Pipeline Health Diagnostician"
  - Input: Pipedrive export + tijdsperiode
